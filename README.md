# brgr-ntbk
Competitive programming notebook for BRGR

## Guidelines
(tk) The following conventions are used:

- Use zero-based indexing. Always.
- All query functions will include the endpoint.
- Weighted adjacency matrices will be stored in the form `adj[u] = {v,w}`, meaning the edge from `u` to `v` has weight `w`.
- If needed, individual (weighted) edges will be stored in the form `{w,u,v}`,
meaning the edge from `u` to `v` has weight `w`.
